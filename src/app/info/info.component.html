What is Simple-CRM?

Simple-CRM is a modern and user-friendly web application, developed using Angular, TypeScript, Material Design and Firebase. It's designed specifically for today's agile business user and provides a set of core functionalities essential for efficient customer relationship management.

How to use it

1) User Management: Create and manage user profiles with just a few clicks.

2) News Feed: Integrated feed that pulls current news from an external API, ensuring you always stay updated.

3) Calendar Integration: Schedule appointments with your clients and colleagues directly within a comprehensive calendar.

4) Platform: Allows users to conduct trades amongst each other, fostering a network of trust and collaboration.

Note: Please do not enter any personal data! All data used in Simple-CRM are for demonstration purposes only!
